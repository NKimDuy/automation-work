# data/output/

## Mục đích
Chứa file kết quả tạo ra sau khi chạy tự động, log xử lý hoặc báo cáo.

## Định dạng mẫu
- `students_lms_sync_{yyyyMMdd}.csv` (kết quả số SV đã sync)
- `error_records_{yyyyMMdd}.csv` (danh sách lỗi)

## Ghi chú
- Xóa file cũ hoặc lưu theo ngày tránh bị ghi đè nếu cần lưu lịch sử.
